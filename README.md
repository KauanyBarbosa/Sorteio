# ✧ Mega Sena ✧
Projeto Educacional Interativo

Projeto educacional interativo desenvolvido para simular o funcionamento do jogo de loteria Mega Sena. A aplicação permite ao usuário gerar sua aposta, realizar o sorteio e conferir os resultados de forma visual, simples e intuitiva, tornando o aprendizado mais dinâmico e acessível.

## ✦ Objetivo

Simular o funcionamento de um jogo de loteria

Aplicar lógica de programação com JavaScript

Trabalhar conceitos de números aleatórios e validação

Unir educação e desenvolvimento front-end

## ✧ Conteúdos

Geração de apostas

Sorteio de 6 números entre 1 e 60

Conferência automática de acertos

Identificação de premiações (Terno, Quadra, Quina e Sena)
