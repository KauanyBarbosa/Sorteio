# ✧Mega Sena

Este projeto foi desenvolvido como uma atividade acadêmica para a ETEC de Cubatão. O objetivo principal é simular o funcionamento de um jogo de loteria (Mega Sena), permitindo ao usuário gerar seus números e conferir o resultado do sorteio em tempo real.

O sistema valida automaticamente a quantidade de acertos e exibe mensagens personalizadas (como "Você acertou o terno!").

## ✧Funcionalidades
Interface Interativa: Painel visual com os números disponíveis.

Geração de Aposta: O usuário pode definir seus números da sorte.

Sorteio Aleatório: Algoritmo que gera 6 números distintos de 1 a 60.

Conferência Automática: Comparação instantânea entre a aposta e os números sorteados.

Feedback Visual: Notificação do tipo de premiação alcançada (Terno, Quadra, Quina ou Sena).
